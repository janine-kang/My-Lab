### HTML / CSS / Java Script / BOOTSTRAP

1. HTML(Hypertext Markup Language)

- 웹페이지가 어떻게 구조화 되어 있는지 브라우저로 하여금 알 수 있도록 하는 마크업 언어. 짜여진 코드=>우리가 익숙한 화면으로 나타내줌
- 웹 컨텐츠의 구조와 의미를 문단, 제목, 표, 삽입 이미지, 동영상 등으로 정의하고 부여

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>바 영역에 나타나는 제목</title>
  </head>
  <body>
    <p>지면에 나타나는 영역</p>
  </body>
</html>
```





2. CSS

- 문서 표현을 위한 스타일시트 언어
- 텍스트 꾸미기, 박스 꾸미기 등 레이아웃 관련된 기술



3. JavaScript

- 동적 다중 패러다임 스크립트 언어. 까다로운 언어이기 때문에 정확하게 문법을 지켜야 동작
- HTML과  CSS가 결합되고 웹페이지 상에서 올려진 후, 브라우저의 자바스크립트 엔진에 의해 실행
- 컨텐츠를 변경하고, 멀티미디어나 움직이는 이미지 등을 다루는 스크립트 언어
- 변수안에 값 저장

```javascript
var a = "a";
var b = "b";
if (a < b) { // true
  console.log(a + " is less than " + b);
} else if (a > b) {
  console.log(a + " is greater than " + b);
} else {
  console.log(a + " and " + b + " are equal.");
}
```
