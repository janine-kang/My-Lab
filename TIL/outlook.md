### 1. 파이썬 

 ##### 1) 라이브러리

> [^표준 라이브러리]: 파이썬 설치 시 자동으로 들어오는 자료형, 내장함수, 표준모듈 등
> [^외부 라이브러리]: 오픈소스로 배포되는 모듈과 패키지를 묶어낸 것. 대표적으로  <u>Pandas, requests, scikit-learn</u> 등이 있음

* `urllib` : URL 작업을 위한 여러 <u>모듈</u>을 모은 <u>패키지</u>

  [^모듈]: 데이터, 함수, 클래스 등이 담겨진 파일.  외부 라이브러리(모듈) 사용 위해서는 따로 import를 하는 작업이 필요. <u>파일 확장자.py</u>
  [^패키지]: 연관된 모듈들의 집합. 아래 정리된 것들의 묶음을 패키지라고 함. 패키지임을 알리기 위해 `__init__.py`로 생성하는 것이 권장된다.
  [^pip]: 파이썬의 패키지 매니저

    * `urllib.request` : URL을 열고 읽음
    * `urllib.error` : `urllib.request`에 의해 발생하는 <u>예외</u>를 포함
    * `**urllib.parse**` : URL 구문 분석
    * `urllib.robotparser` : robots.txt 파일 구문 분석 위한 



##### 2) 웹 크롤링

> 브라우저에서 보는 정보를 내 입맛에 맞게 가공하여 로컬에 저장하는 행위

* Requests : HTTP를 호출하여, html 불러옴
* BeautifulSoup 
  * requests 라이브러리로 http를 호출하고 난 후, html 코드를 파이썬이 이해하는 객체 구조로 변환(parsing)
  * requests는 객체 구조로 데이터를 만들 수 없기 때문에 사용
  * BeautifulSoup을 직접 작성하여 설치하는 것 보다, bs4라는 wrapper라이브러리를 통해 설치하는 방법이 더 쉽고 안전
