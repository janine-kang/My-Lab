# 기초 개념 정리

> 상황이 어떻게 돌아가는지 알아야 하니까 



`HEAD` : 지금 작업하는 로컬 브랜치 

`master` : default로 생성되는 branch

`pull` : fetch의 역할에 merge를 더해 실행

`fetch` : 원격저장소에 있는 내용을 불러오기, merge를 하지 않음

​	- `FETCH_HEAD` : 원격 저장소의 가장 최신 commit 이력 존재

`리모트 저장소` : 인터넷이나 네트워크 어딘가에 있는 저장소. Repository의 URL



커밋했을 때 아래에 어떤 사항이 바뀌었는지 내역이 주르륵 뜸!
